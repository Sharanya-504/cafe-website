# Cafe Website

This is a simple **cafe website** built using **HTML, CSS, and Bootstrap**.  
It showcases the cafe's menu, home page, and styling using Bootstrap components.

## Pages

- `index.html` - Home page
- `menu.html` - Menu page
- `style.css` - Custom styles
- `images/` - Images used in the website

## Features

- Responsive design using **Bootstrap**
- Clean layout for cafe home page and menu
- Easy to customize for any cafe
